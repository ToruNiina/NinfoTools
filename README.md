##C++ CafeMol ninfo tool

CafeMolが出力するninfoファイルを取り扱うためのツール（まだ完全でないsplitterしかないが）。
必要性を感じるか、感じなくても気が向いたら更新する予定。

liberica is a name of one of the coffee trees.

###Reference
